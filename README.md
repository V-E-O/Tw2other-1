# 安装

* 修改config目录中的config.inc.php.sample为config.inc.php,用编辑器打开修改后的文件，按照文件中的提示填写必须信息,不需要同步的目标留空即可。注意ENCRYPT_SECRET的长度必须为8的倍数。

* 运行install.php，按照提醒一步一步走，直到完成。


# 运行

配置系统Cron，调用cron.php，完工。

# 注意
